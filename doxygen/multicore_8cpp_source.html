38 /*
39 Now that threads have taken over the actual_v, v_node, etc, it might
40 be a good time to regularize the method of freeing, allocating, and
41 updating those arrays. To recapitulate the history, Node used to
42 be the structure that held direct values for v, area, d, rhs, etc.
43 That continued to hold for the cray vectorization project which
44 introduced v_node, v_parent, memb_list. Cache efficiency introduced
45 actual_v, actual_area, actual_d, etc and the Node started pointing
46 into those arrays. Additional nodes after allocation required updating
47 pointers to v and area since those arrays were freed and reallocated.
48 Now, the threads hold all these arrays and we want to update them
49 properly under the circumstances of changing topology, changing
50 number of threads, and changing distribution of cells on threads.
51 Note there are no longer global versions of any of these arrays.
52 We do not want to update merely due to a change in area. Recently
53 we have dealt with diam, area, ri on a section basis. We generally
54 desire an update just before a simulation when the efficient
55 structures are necessary. This is reasonably well handled by the
56 v_structure_change flag which historically freed and reallocated
57 v_node and v_parent and, just before this comment,
58 ended up setting the NrnThread tml. This makes most of the old
59 memb_list vestigial and we now got rid of it except for
60 the artificial cells (and it is possibly not really necessary there).
61 Switching between sparse and tree matrix just cause freeing and
62 reallocation of actual_rhs.
63 
64 If we can get the freeing, reallocation, and pointer update correct
65 for _actual_v, I am guessing everything else can be dragged along with
66 it. We have two major cases, call to pc.nthread and change in
67 model structure. We want to use Node* as much as possible and defer
68 the handling of v_structure_change as long as possible.
69 */

115 void nrn_mk_table_check() {
116  if (table_check_) {
117  free((void*)table_check_);
118  table_check_ = nullptr;
119  }
120  auto& memb_func = corenrn.get_memb_funcs();
121  // Allocate int array of size of mechanism types
122  std::vector<int> ix(memb_func.size(), -1);
123  table_check_cnt_ = 0;
124  for (int id = 0; id < nrn_nthread; ++id) {
125  auto& nt = nrn_threads[id];
126  for (auto tml = nt.tml; tml; tml = tml->next) {
127  int index = tml->index;
128  if (memb_func[index].thread_table_check_ && ix[index] == -1) {
129  ix[index] = id;
130  table_check_cnt_ += 2;
131  }
132  }
133  }
134  if (table_check_cnt_) {
135  table_check_ = (ThreadDatum*)emalloc(table_check_cnt_ * sizeof(ThreadDatum));
136  }
137  int i = 0;
138  for (int id = 0; id < nrn_nthread; ++id) {
139  auto& nt = nrn_threads[id];
140  for (auto tml = nt.tml; tml; tml = tml->next) {
141  int index = tml->index;
142  if (memb_func[index].thread_table_check_ && ix[index] == id) {
143  table_check_[i++].i = id;
144  table_check_[i++]._pvoid = (void*)tml;
145  }
146  }
147  }
